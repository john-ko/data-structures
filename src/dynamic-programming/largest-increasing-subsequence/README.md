# largest increasing subsequence

*given*: [4, 2, 5, 8, 1]

largest increasing subsequence: 3 [2, 5, 8] || [4, 5, 1]

deals "cards" to top most to bottom most
```
[4, 2, 1]
[5]
[8]
```
